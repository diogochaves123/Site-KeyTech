# 🚀 KeyTech - Site Institucional

Um site moderno, dinâmico e responsivo para a empresa KeyTech, desenvolvido com foco em experiência do usuário e design impactante.

## ✨ Características

### 🎨 Design
- **Paleta de cores personalizada**: Verde/azul escuro (40%), Cinza (30%), Preto (20%), Branco (10%)
- **Design responsivo**: Otimizado para desktop, tablet e mobile
- **Animações suaves**: Transições e efeitos visuais modernos
- **Tipografia moderna**: Fonte Inter para melhor legibilidade

### 🛠️ Funcionalidades
- **Navegação suave**: Scroll suave entre seções
- **Chat widget interativo**: Simulação de atendimento ao cliente
- **Formulário de contato**: Validação em tempo real
- **Animações de entrada**: Elementos aparecem conforme o scroll
- **Contadores animados**: Estatísticas com animação
- **Menu mobile**: Hamburger menu responsivo
- **Scroll to top**: Botão para voltar ao topo

### 📱 Seções do Site
1. **Header**: Navegação fixa com logo e menu
2. **Hero**: Seção principal com título animado e CTA
3. **Serviços**: Cards interativos com hover effects
4. **Destaque**: Seção com showcase de recursos
5. **Sobre**: Informações da empresa com estatísticas
6. **CTA**: Call-to-action para contato
7. **Contato**: Formulário e informações de contato
8. **Footer**: Links úteis e informações da empresa

## 🚀 Como Usar

### Pré-requisitos
- Navegador web moderno
- Servidor web (opcional para desenvolvimento local)

### Instalação
1. Clone ou baixe os arquivos do projeto
2. Certifique-se de que a logo está na pasta `img/` com o nome `KeyTech.png`
3. Abra o arquivo `index.html` no navegador

### Estrutura de Arquivos
```
Site KeyTech/
├── index.html          # Estrutura HTML principal
├── styles.css          # Estilos CSS
├── script.js           # Funcionalidades JavaScript
├── README.md           # Documentação
└── img/
    └── KeyTech.png     # Logo da empresa
```

## 📱 Responsividade

O site é totalmente responsivo e se adapta a diferentes tamanhos de tela:

- **Desktop**: Layout completo com todas as funcionalidades
- **Tablet**: Ajustes de grid e espaçamentos
- **Mobile**: Menu hamburger, layout em coluna única

### Breakpoints
- `768px`: Tablet
- `480px`: Mobile

## 🔧 Personalização

### Cores
Para alterar as cores, edite as variáveis CSS no arquivo `styles.css`:

```css
:root {
    --primary-color: #sua-cor-principal;
    --secondary-color: #sua-cor-secundaria;
    /* ... outras cores */
}
```

### Conteúdo
- **Textos**: Edite diretamente no arquivo `index.html`
- **Imagens**: Substitua as imagens na pasta `img/`
- **Informações de contato**: Atualize no HTML

### Funcionalidades
- **Chat**: Personalize as respostas no arquivo `script.js`
- **Formulário**: Configure validações e envio
- **Animações**: Ajuste velocidades e efeitos

## 🐛 Solução de Problemas

### Problemas Comuns
1. **Logo não aparece**: Verifique se o arquivo está em `img/KeyTech.png`
2. **Animações não funcionam**: Verifique se o JavaScript está carregado
3. **Layout quebrado**: Verifique se todos os arquivos CSS estão carregados
4. **Chat não abre**: Verifique o console do navegador para erros

### Debug
- Abra o console do navegador (F12)
- Verifique se há erros JavaScript
- Teste em diferentes navegadores
- Verifique a responsividade em diferentes dispositivos
